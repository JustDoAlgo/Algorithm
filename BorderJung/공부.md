## Map vs Unordered Map

1. 내부 구현 차이
- Map: 레드-블랙 트리(균형 이진 탐색 트리)
    - 범위 기반 탐색이 필요하다면 사용
- UnorderedMap: 해시 테이블
    - 빠른 조회가 필요할 때 사용

2. 정렬
- Map: 키 기준 자동 정렬
- UnorderedMap: 정렬 x

3. 시간 복잡도
Search, Insert, Delete에 대해
- Map: O(log N)
- UnorderedMap: O(1) ~ O(N)
