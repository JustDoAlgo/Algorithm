### Q2. getline
다음 코드에서 입력이 `"hello world"`일 때, `s`에 저장되는 값은?
```cpp
string s;
cin >> s;
```

"hello"

<details>
<summary>정답</summary>

`"hello"`만 저장된다. `cin >>`은 공백에서 끊기므로, 공백 포함 한 줄 입력은 `getline(cin, s)`를 사용해야 한다.
</details>

---

### Q3. getline 주의점
다음 코드의 문제점은?
```cpp
int n;
cin >> n;
string s;
getline(cin, s);
```

n 입력 받은 후에 \n이 남아있어서, getline은 새로운 라인을 받는게 아니라 빈 문자열을 읽어버림
cin.ignore(); 을 추가하면 \n을 제거할 수 있음

<details>
<summary>정답</summary>

`cin >> n` 이후 버퍼에 `\n`이 남아있어서, `getline`이 빈 문자열을 읽는다.
해결: `cin >> n` 뒤에 `cin.ignore()`를 추가한다.
```cpp
int n;
cin >> n;
cin.ignore();
string s;
getline(cin, s);
```
</details>

### Q4. string -> int
문자열 `"12345"`를 `int`로 변환하는 코드를 작성하세요. (2가지 방법)

int i = stoi("12345");
int형 외에도 여러가지로 형 변환 가능. stol, stoll, stof, stod

stringstream을 사용하는 방법
s = "12345";
stringstream ss(s);
int num; ss >> num;

<details>
<summary>정답</summary>

```cpp
// 방법 1: stoi
int num = stoi("12345");

// 방법 2: stringstream
string s = "12345";
stringstream ss(s);
int num; ss >> num;
```
- 관련 함수: `stol`(long), `stoll`(long long), `stof`(float), `stod`(double)
- 큰 수는 `stoll` 사용 (stoi 범위 초과 시 예외 발생)
</details>

string s = "aaaa";

### Q11. rfind
`rfind()`는 `find()`와 어떻게 다른가? 활용 예시를 하나 드세요.

뒤에서 부터 찾고 싶으면 rfind() 사용.

<details>
<summary>정답</summary>

`rfind()`는 **뒤에서부터** 탐색하여 마지막으로 등장하는 위치를 반환한다.

```cpp
}
```
</details>

### Q17. compare 메서드
다음 두 코드의 차이는?
```cpp
// A
if (a == b.substr(0, a.size()))
// B
if (b.compare(0, a.size(), a) == 0)
```

<details>
<summary>정답</summary>

기능은 동일하지만, **B가 더 효율적**이다.
- A: `substr`이 새 문자열을 **복사 생성**한다.
- B: `compare`는 **제자리에서** 부분 문자열을 비교한다 (복사 없음).
- `compare(pos, len, str)`: 같으면 0 반환.
</details>

### Q19. 와일드카드 매칭
패턴 `"abc*xyz"`로 파일명을 매칭할 때, `"abxyz"`는 매칭되는가? 이유는?

<details>

</details>

### Q20. 종합 코드
크로아티아 알파벳 문제에서, 다중 패턴을 한 글자로 치환하는 접근법의 핵심 아이디어는?

<details>
<summary>정답</summary>

각 패턴을 `find()`로 찾아 `replace()`로 한 글자(`"#"`)로 치환 → 최종 문자열 길이가 답.
```cpp
string patterns[] = {"c=", "c-", "dz=", "d-", "lj", "nj", "s=", "z="};
for (auto& p : patterns) {
    size_t pos = 0;
    while ((pos = s.find(p, pos)) != string::npos) {
        s.replace(pos, p.size(), "#");
        pos += 1;
    }
}
cout << s.size();
```
- 복잡한 visited 관리 대신 **치환으로 단순화**하는 발상 전환이 핵심.
</details>

## 빠른 체크리스트

| 메서드 | 시그니처 | 반환값 |
|--------|----------|--------|
| `find` | `s.find(sub, pos)` | 첫 인덱스 / `npos` |
| `rfind` | `s.rfind(sub)` | 마지막 인덱스 / `npos` |
| `substr` | `s.substr(pos, len)` | 새 문자열 |
| `replace` | `s.replace(pos, len, newStr)` | `s&` (자기 참조) |
| `compare` | `s.compare(pos, len, str)` | 0(같음), 양/음수 |
| `stoi` | `stoi(s)` | int |
| `to_string` | `to_string(num)` | string |
